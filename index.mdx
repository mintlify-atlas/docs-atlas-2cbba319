## What is Rudi Foodi?

Rudi Foodi started as a simple prototype with one room and a few drifting treats. After each update — Zoomies, cloning, power-ups, size modes, dancing, meteors, flying debris — things got progressively stranger. What emerged is a 20-level browser game driven by treats, friendship, and absolute nonsense.

You play as Rudi, navigating through increasingly chaotic themed worlds. Collect enough treats to advance, grab power-ups along the way, and try to survive meteor strikes and clone invasions long enough to reach the Final Dog Park.

## Quick Start

<Steps>
  <Step title="Get the source">
    Clone or download the repository from [GitHub](https://github.com/mbeckham4-hub/Rudi-Foodi).
  </Step>
  <Step title="Open the game">
    Open `rudi_low_poly_3_d_game 6.html` directly in any modern browser — no server or build step needed.
  </Step>
  <Step title="Play">
    Click **Play** on the title screen and start collecting treats. Reach the treat goal for each level to advance.
  </Step>
  <Step title="Hack it">
    Edit the `levelThemes` or `musicThemes` arrays in the HTML file to customize worlds and music. See the [Developer Guide](/dev/setup) for details.
  </Step>
</Steps>

<Note>
  Rudi Foodi is fully open source under its repository license. Contributions, forks, and new level ideas are welcome.
</Note>
